# Computer Graphics Assignment: Flappy Bird 2D Game
## Department of Computer Science

**Project Description:**
This project is an interactive 2D side-scrolling game developed using the HTML5 Canvas API and JavaScript. It demonstrates core computer graphics concepts including rendering, physics (gravity and collision detection), animation loops, and parallax background scrolling.
